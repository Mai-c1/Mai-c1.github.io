# GSOJ 简介

**GSOJ**，是浙江广厦大学程序设计竞赛团队开发的基于SpringBoot+Vue的分布式架构的在线评测系统。

## 什么是OJ？

OJ（Online Judge，在线评测系统）是一个在线平台，用户可以在上面提交代码来解决各种编程问题。系统会自动运行用户的代码，检查它是否能正确解决问题，并评估其效率。每个问题都会提供一些输入数据和预期的结果，代码需要在规定的时间和内存限制内给出正确的答案。

简单来说，OJ就像是一个在线的编程练习场，它能帮你检验自己的代码是否正确，同时也是一个提升编程技巧的好地方。

## 适合人群

- **初学者**：刚刚开始学习编程的人可以通过OJ上的简单题目来练习基础语法和逻辑思维。
- **竞赛教练**：利用OJ作为训练工具，帮助团队成员准备各类编程比赛，定制训练计划。GSOJ提供的爬虫服务能自动收集团队成员在Codeforces、力扣等平台上的关键表现数据，包括通过题数、Rating分数和周赛排名。这使得教练能快速了解成员的进步情况，制定更有效的训练策略。
- **长期参与编程竞赛的同学**：GSOJ是一个值得学习的入门项目，可以从中学习到前后端开发主流的技术栈以及运维相关的知识。
